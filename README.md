# Privacy Policy – PRIZEO


PRIZEO values your privacy. This Privacy Policy explains how data is collected, used, and protected when you use the PRIZEO Discord giveaway bot.

---

## 1. Information We Collect
PRIZEO may collect and store limited data required for proper functionality, including:
- User IDs
- Server (Guild) IDs
- Giveaway data (entries, winners, settings)
- Command usage and configuration data

We do **not** collect personal information such as emails, passwords, IP addresses, or messages.

---

## 2. How We Use Your Data
Collected data is used only to:
- Operate and manage giveaways
- Ensure fair winner selection
- Maintain bot functionality
- Improve bot features and performance

---

## 3. Data Sharing
PRIZEO does **not**:
- Sell user data
- Share data with third parties
- Use data for advertising or tracking

Data is used strictly for internal bot operations.

---

## 4. Data Storage & Security
- Data is stored securely
- Only the bot developer has access to stored data
- Reasonable measures are taken to prevent unauthorized access

---

## 5. Data Retention
- Data is retained only as long as necessary for bot functionality
- Data may be automatically removed if the bot is removed from a server
- You may request data removal by contacting the developer

---

## 6. User Rights
You have the right to:
- Request what data is stored about your server or user ID
- Request deletion of stored data (where applicable)

Requests can be made via the contact details below.

---

## 7. Children’s Privacy
PRIZEO does not knowingly collect data from users under the age of 13. Usage of the bot must comply with Discord’s age requirements.

---

## 8. Changes to This Policy
This Privacy Policy may be updated at any time. Continued use of PRIZEO after changes means you accept the updated policy.

---
